Overview

The Quantum-Enhanced Simulation Suite is a comprehensive CAE (Computer-Aided Engineering) platform that brings quantum computing capabilities to physics simulations. Built during my internship at Ansys Research (Synopsys), this project demonstrates how quantum algorithms can accelerate and enhance traditional simulation workflows.

The platform supports two primary simulation modalities:

  • Electromagnetic (EM) Scattering ‚Äì Time-series field evolution computed via Hamiltonian simulation
  • Quantum Lattice Boltzmann Method (QLBM) ‚Äì Fluid dynamics with interactive 3D density visualization

Both workflows leverage quantum computing backends (IBM Quantum, IonQ) while maintaining fallback to classical simulators, making the platform accessible for demonstration and education even without QPU access.

System Architecture

The application follows a modular architecture with a unified Trame server managing both EM and QLBM workflows. The design prioritizes separation of concerns while enabling code reuse for common quantum operations.

Application Flow

The main application (app.py) acts as the entry point, initializing the Trame server and loading both simulation modules. Each module manages its own state, UI components, and quantum circuit construction.

EM Scattering Simulation

The electromagnetic module implements Hamiltonian simulation for computing scattered field evolution. The physics is encoded in a Hamiltonian operator, and time evolution is performed using Trotter-Suzuki decomposition on quantum hardware.

Workflow

  1. Geometry Definition ‚Äì Define scattering geometry and excitation parameters
  2. Hamiltonian Construction ‚Äì Build the electromagnetic Hamiltonian operator
  3. Circuit Generation ‚Äì Trotterize the time evolution operator
  4. Variational Compression ‚Äì Optionally compress deep circuits using adaptive ansatz
  5. Execution ‚Äì Run on selected backend (simulator or QPU)
  6. Visualization ‚Äì Plot time-series expectation values

Technical Highlights

  • Implements first and second-order Trotter decomposition
  • Variational circuit compression achieves 99.7% depth reduction
  • Supports multiple observables for field component extraction
  • Real-time circuit statistics (depth, gate count, 2Q gates)

QLBM Fluid Simulation

The Quantum Lattice Boltzmann Method module implements fluid dynamics simulation using quantum computing. The LBM algorithm naturally maps to quantum operations, with streaming and collision steps encoded as quantum gates.

Workflow

  1. Grid Setup ‚Äì Initialize lattice dimensions and boundary conditions
  2. State Encoding ‚Äì Encode fluid distribution functions in qubits
  3. Collision Step ‚Äì Apply BGK collision operator
  4. Streaming Step ‚Äì Propagate distributions to neighboring cells
  5. Measurement ‚Äì Extract macroscopic quantities (density, velocity)
  6. 3D Visualization ‚Äì Render density field using PyVista
