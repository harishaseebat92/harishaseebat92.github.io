Overview

The Vehicle Routing Problem (VRP) asks for optimal routes for a fleet of vehicles to serve a set of customers while minimizing total distance/cost under constraints (capacity, visit-once, optional time windows, etc.). It’s a central problem in operations research and logistics—and a good candidate to explore QUBO/QAOA-style formulations.

Formulation

  • Objective: minimize total travel distance (or cost)
  • Constraints: each customer visited once; depot start/end; vehicle capacity; sub-tour elimination; (optional) time windows
  • Common variants: CVRP, VRPTW

For quantum-inspired approaches, the constraints are encoded into a QUBO using penalty terms.

Classical baselines

  • OR-Tools: routing solver + local search metaheuristics for practical performance
  • Pyomo: MILP formulation for smaller instances / exact baselines

Quantum-inspired approaches

  • QUBO encoding: map routing decisions + constraints into binary variables
  • Quantum Annealing: solve QUBO on annealers (or classical annealing samplers)
  • QAOA: variational circuit built from problem Hamiltonian + mixer with classical parameter optimization
